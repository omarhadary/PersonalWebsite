Tukio

Tukio is an app that allows authenticated users to search event tickets through the Eventful API, save events of their choice and commit to buy discounted tickets for those events. A progress bar dynamically updates as users commit to buy tickets for the events.

Follows an MVC design pattern. Uses React to render HTML. Uses Node.js and MYSQL/Sequelize to query data. Uses Webpack module bundler.

Latest News

Latest-News is a full stack web app that allows users save and add add comments to NPR news articles. Articles are populated using Cheerio and saved to a Mongo DB.

Follows an MVC design pattern. Uses Node.js/Express Framework server. MongoDB with Mongoose ORM library to query and route data. Uses Handlebars to generate HTML. Heroku for deployment.
